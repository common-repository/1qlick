The best one click checkout experience brought to you by 1Qlick.

== Installation ==

Please unzip the archive and upload the result to wp-content/plugins/1qlick

== Configuration ==

Go to WordPress admin -> WooCommerce -> Settings -> 1Qlick settings

Select the hooks to which you would like to add the *Checkout with 1Qlick* button. `woocommerce_after_cart_totals`
is advised, but custom theme's might not implement it. In that case, please select another hook.

The environment setting allows you to test the integration with 1Qlick. Please set the environment settings to `testing`
to enable test mode.
